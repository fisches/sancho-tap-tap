# Sancho Tap Tap

Mini appli web locale pour enfant: on tape partout, des emojis apparaissent.

## Lancer

Option simple:

```bash
python3 -m http.server 8000
```

Puis ouvrir `http://localhost:8000`.

## Notes

- Aucun framework ni dépendance.
- Le fond change à chaque rechargement.
- L'appli fonctionne à la souris et au tactile.
